Protocole graphique Théo NICOLAS G3

Pour lancer correctement le programme :

- ouvrir le fichier "Hexa_circle.html" dans n'importe quel navigateur web
- tapez votre message dans la zone de texte
- Cliquer sur le bouton "Encoder" pour lancer l'encodage du message OU sur le bouton "Encodage Simple" pour lancer l'encodage sans détection d'Erreur
- Pour relancer une deuxième fois recharger la page => F5 ou cmd + R sur mac (si cela ne marche pas => CTRL + F5)

Conseil : Pour mieux comprendre lancer avec l'encodage simple avec un message court comme "a" qui correspond à 61 en hexa
